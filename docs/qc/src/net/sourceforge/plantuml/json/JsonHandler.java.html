28   /**
29    * A handler for parser events. Instances of this class can be given to a {@link JsonParser}. The
30    * parser will then call the methods of the given handler while reading the input.
31    * <p>
32    * The default implementations of these methods do nothing. Subclasses may override only those
33    * methods they are interested in. They can use <code>getLocation()</code> to access the current
34    * character position of the parser at any point. The <code>start*</code> methods will be called
35    * while the location points to the first character of the parsed element. The <code>end*</code>
36    * methods will be called while the location points to the character position that directly follows
37    * the last character of the parsed element. Example:
38    * </p>
39    *
40    * <pre>
41    * ["lorem ipsum"]
42    *  ^            ^
43    *  startString  endString
44    * </pre>
45    * <p>
46    * Subclasses that build an object representation of the parsed JSON can return arbitrary handler
47    * objects for JSON arrays and JSON objects in {@link #startArray()} and {@link #startObject()}.
48    * These handler objects will then be provided in all subsequent parser events for this particular
49    * array or object. They can be used to keep track the elements of a JSON array or object.
50    * </p>
51    *
52    * @param <A>
53    *          The type of handlers used for JSON arrays
54    * @param <O>
55    *          The type of handlers used for JSON objects
56    * @see JsonParser
57    */
58   public abstract class JsonHandler<A, O> {

81      */
82     public void endNull() {
